## 확인 항목

1. **구현 현황**: 각 모듈(Backend/Frontend/App)별 구현된 기능 파악
   - Backend: 컨트롤러, 서비스, 도메인 모델 파일 스캔
   - Frontend: 페이지, 컴포넌트 파일 스캔
   - App: Flutter 위젯 파일 스캔

2. **테스트 현황**: 테스트 파일 수와 커버리지 확인
   ```bash
   cd Backend && ./gradlew test 2>&1 | tail -20
   ```

3. **PROJECT_PLAN.md 기준 진행률**: 기획안의 Sprint 목표 대비 현재 상태

4. **남은 작업 목록**: 미구현 기능 리스트업
